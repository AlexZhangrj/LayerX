LayerX

An intuitive app to display transparent images on screen.



Download v1.2.0

Features

  1. Drag and drop images
  2. Keyboard Shortcuts
  3. Adjust transparency
  4. Lock images
  5. Aspect ratio scaling

Easy to use

Simply drag and drop on the placeholder, there you go. is that easy enough?

Keyboard Shortcuts

Key Action
⌘ 0 Acutal image size.
⌘ + Scale up 10%
⌘ - Scale down 10%
^ ⌘ + Scale up 1px
^ ⌘ - Scale down 1px
⌘ J Increase transparency
⌘ K Reduce transparency
⌘ L Lock the image and make window always on top
↑ Move up 1px.
← Move left 1px.
→ Move right 1px.
↓ Move down 1px.

Mouse Events

Event Action
Scroll up on image Reduce transparency
Scroll down on image Increase transparency
